Sonic Kit PvP

     

This is the kit PvP map made by Popularsonic(me), the map has 3 kits(I will add more If I want to). This map is made out of complex Commands, enjoy my first Kit PvP map

This map supports versions 1.8 and above

The map has a spectate portal that, when entered, takes you to the spectator area, and a door which opens when you get close enough, name tags to identify Kits and is also used for navigation. This map will get updated over time, again, enjoy the Kit PvP map!

Installation

iOS 12 and below- Download the .mcworld file and open it in Minecraft

iOS 13- Download the zip file to the files app, unzip the file and move the folder to On my iPad/Minecraft/games/com.mojang/worlds folder
